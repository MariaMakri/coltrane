frontiers2016_orig/ contains the original source code and output from model version 1.0 that goes with Banas et al., Front. Mar. Sci., 2016. Since then, the output structures have been heavily reorganised, so this code will not run!

frontiers2016/ contains selected cases from Banas et al., Front. Mar. Sci., 2016 in the current model version (i.e., this code will actually run).
